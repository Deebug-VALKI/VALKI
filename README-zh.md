## 📖 產品簡介 (Overview)

**VALKI** 是由 **DEEBUG** 團隊專為《特戰英豪》(VALORANT) 全球社群打造的高效行動助手。我們打破了 PC 與現實生活的隔閡，讓你在通勤、休息或任何時刻，都能隨時同步你的戰績、皮膚商店與競技狀態。

無論是查看今日的**每日商店**、分析上一場比賽的**數據表現**，或是參與**皮膚評分**，VALKI 都能讓你在手機上掌握所需的一切資訊。

> [!NOTE]
> **VALORANT** 是一款 PC 遊戲。**VALKI** 是第三方開發的行動應用程序，旨在透過數據視覺化與遠端工具提升玩家的遊戲體驗。

## 🔒 安全與免責聲明

我們極度重視您的帳戶安全，在使用 VALKI 前請務必注意：

* **官方聲明**：VALKI 非 Riot Games 官方附屬、授權或贊助之應用程式。
* **API 使用**：由於部分功能（如商店數據）使用了非公開 API 接口，使用過程中存在第三方工具常見的潛在風險。
* **安全性建議**：強烈建議您的 Riot 帳戶開啟 **雙重驗證 (2FA)**，以確保在使用任何第三方服務時擁有最高等級的安全保障。
